FAQ

Common questions

Here are some of the most common questions that we get.
What is the purpose of Gratiforge Gratiforge is a web application which allows users to generate gratitude letters from a spreadsheet of attendee information.
What types of data can users enter? The parameters a user can specify depend on their selected plan. Basic users can enter an event name, and the name of the signee of each letter. Pro users can enter the basic parameters, and can save or upload their own parameterized templates. Enterprise users can use the basic and pro features, and can specify the structure of their data rather than modifying it to fit the standard structure of the app.
How can users upload a spreadsheet? Users can upload a spreadsheet by clicking on the 'Upload' button and selecting the file from their device.
What file formats are supported for uploading? Any spreadsheets in .xlsx format are supported for uploading.
How many generated files will be included in the downloaded zip archive? The number of generated files included in the downloaded zip archive will depend on the data in the spreadsheet. Each row in the user's spreadsheet should represent one recipient entity, and thus one letter.
